# Dev

1. Clonar el proyecto
2. Copiar el ```env.template``` y renombrar a ```.env```
3. Ejecutar
```
yarn install
```
4. Levantar la imagen (Docker Desktop)
```
docker-compose up -d
```
5. Levantar backend de Nest
```
yarn start:dev
```
6. Visitar el sitio
```
localhost:3000/graphql
```
